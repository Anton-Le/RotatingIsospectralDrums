Set-up of the damping (or transport) matrix.

Parameters
[in,out]CUncompressed and initialized matrix in CRS/CSR format.
[in]geoStruct of type geometry2 containing the geometry data of the domain.
[in]GDa nx3 array containing in it's rows the nodes and weights of the gauss quadrature.
[in]QuadOrderNumber of nodes of the quadrature (# of rows of GD).
[in]freeNodesNumber of free (not bound by Dirichlet conditions) nodes of the mesh.
[in]NnzPerRowEigen integer vector of length = number of free nodes in the grid.
Returns
Matrix C filled with the entries of the discretised \(\partial_\varphi \) vector field.

The function fills the provided matrix with entries of a FEM gyroscopic matrix. The matrix is a discrete version of an azimuthal unit vector field in the real plane in polar coordinates and assumes Dirichlet boundary conditions. The order of the ansatz functions and the number of associated nodes per triangle is given globally using PkELEMENTS preprocessor directive.

Set-up of the full mass matrix.

Parameters
[in,out]MUncompressed and initialized matrix.
[in]geoStruct of type geometry2 containing the geometry data of the domain.
[in]GDa nx3 array containing in it's rows the nodes and weights of the gauss quadrature.
[in]QuadOrderNumber of nodes of the quadrature (# of rows of GD).
[in]freeNodesNumber of free (not bound by Dirichlet conditions) nodes of the mesh.
[in]NnzPerRowEigen integer vector of length = number of free nodes in the grid.
[in]omegaAngular frequency of the domain.
[in]IORIndex of refraction of the medium the domain is comosed of.
Returns
Filled mass matrix M.

The function fills a matrix M in CSR/CSC format with entries corresponding to a discrete version of the full mass matrix ( \( \gamma\cdot\left( n^2-\left(\frac{r\omega}{c}\right)^2\right) \)). For the mass matrix no assumptions as to the type of the boundary conditions are necessary.

The entries depend, of course, on the order of the ansatz polynomials \( \mathcal{P}_k \).
