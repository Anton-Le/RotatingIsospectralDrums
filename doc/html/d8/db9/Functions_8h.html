Set-up of the damping (or transport) matrix.

Parameters
[out]CUncompressed and initialized matrix in CRS/CSR format.
[in]geoStruct of type geometry2 containing the geometry data of the domain.
[in]GDa nx3 array containing in it's rows the nodes and weights of the gauss quadrature.
[in]QuadOrderNumber of nodes of the quadrature (# of rows of GD).
[in]freeNodesNumber of free (not bound by Dirichlet conditions) nodes of the mesh.
[in]NnzPerRowEigen integer vector of length freeNodes .

The function fills the provided matrix C with entries of a FEM gyroscopic matrix. The matrix is a discrete version of an azimuthal unit vector field \(\partial_\varphi \) in the real plane. It implements eq. (6.12c) from the thesis.

Though Dirichlet boundary conditions are not explicitly assumed in the function, its form is nevertheless indicative of this assumption seeing as terms implementing necessary boundary integrals are missing.

The order of the ansatz functions and the number of associated nodes per triangle is given globally using PkELEMENTS preprocessor directive.

The parameters QuadOrder and GD are used to pass the data necessary for an implementation of a numerical quadrature and NnzPerRow and freeNodes are used for resizing and efficient memory handling.

On exit the matrix C is overwritten with new entries.

Mapping from I=[-1,1] to \( \mathbb{R}^2 \).

Parameters
[in]iIndex (global) of the starting (left) point of the line.
[in]jIndex (global) of the ending (right) point of the line.
[in]sLocal coordinate on I=[-1,1].
[in]*xPointer to the array containing all the vertices of the triangulation.
[out]yEigen vector to which the result of the mapping is stored.
Returns
y 2D vector with y = 1/2 * (x[j] - x[i])*s + 1/2 *(x[j] + x[i]).

The function computes the mapping from I=[-1,1] to R^2 for a parametrized edge starting at x[i] and ending at x[j]. On exit the global coordinates (x,y) of the point with the edge-coordinate s are stored in the first and second component of y.

The mapping is:

\[ y = \frac{1}{2}\left( x[j] -x[i] \right) s + \frac{1}{2}\left( x[j]+ x[i] \right). \]

Set-up of the mass matrix.

Parameters
[out]MUncompressed and initialized matrix in CSR/CRS format.
geoStructure of type geometry2 containing the geometry of the domain.
[in]GDA nx3 array containing in its rows the nodes and weights of the gauss quadrature.
[in]QuadOrderNumber of nodes of the quadrature (#rows of GD).
[in]freeNodesNumber of free (non-Dirichlet) nodes in the mesh.
[in]NnzPerRowEigen integer vector of length freeNodes. A vector containing the estimate for the number of non-zeros per row of the matrix.

The function fills a given matrix M with entries of a mass matrix of a FEM. The routine requires a Gauss-type quadrature in 2D provided by a list GD of weights and nodes along with the number QuadOrder of these. The number of freeNodes is used for the resizing of the matrix and NnzPerRow contains the number of non-zero entries per row which allows a more efficient filling of the matrix by avoiding constant resize-and-copy operations. The matrix M is resized and overwritten.

Set-up of the full mass matrix.

Parameters
[out]MUncompressed and initialized matrix.
[in]geoStruct of type geometry2 containing the geometry data of the domain.
[in]GDA nx3 array containing in it's rows the nodes and weights of the gauss quadrature.
[in]QuadOrderNumber of nodes of the quadrature (# of rows of GD).
[in]freeNodesNumber of free (not bound by Dirichlet conditions) nodes of the mesh.
[in]NnzPerRowEigen integer vector of length = number of free nodes in the grid.
[in]omegaAngular frequency of the domain.
[in]IORIndex of refraction of the medium the domain is comosed of.

The function fills a matrix M in CSR/CSC format with entries corresponding to a discrete version of the full mass matrix ( \( \gamma\cdot\left( n^2-\left(\frac{r\omega}{c}\right)^2\right) \)). For the mass matrix no assumptions as to the type of the boundary conditions are necessary. The function implements eq. (6.12d) of the accompanying thesis.

Again, QuadOrder and GD are used for the implementation of a numerical quadrature and freeNodes and NnzPerRow for memory handling.

The angular frequency parameter omega and the index of refraction IOR are necessary to compute the coefficient which scales the whole matrix.

The entries depend, of course, on the order of the ansatz polynomials \( \mathcal{P}_k \).
