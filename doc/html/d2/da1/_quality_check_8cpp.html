Computes the approx. L^2 deviation of the gradients of a numerical solution from an analytic one.

Function which computes an approximation of the L^2 error $|| u- u_{h}||_{L^2}$ of a numerical solution using the analytic solution and Gauss quadrature to approximate the integrals. NOTE! It is vital that the quadrature is of sufficiently high order due to the occurence of products of polynomials.

Parameters
vals[in] Vector containing the computed values of the approximation.
geo[in] Struct of type geometry2 containing the mesh of the domain.
GD[in] Array of nodes and weights of Gauss quadrature.
QuadOrder[in] Number of nodes of the quadrature
Returns
Err L^2 error approximation.
